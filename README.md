## 🚀 Project Overview
This project presents a fully interactive Excel Dashboard designed to analyze and visualize sales performance data in a clear, dynamic, and decision-focused way. The dashboard transforms raw business data into meaningful insights using charts, slicers, and visual analytics.
The main objective of this project is to demonstrate data visualization, dashboard design, and analytical skills using Microsoft Excel as a Business Intelligence (BI) tool.

## 🎯 Project Objectives
•	Convert raw data into actionable insights.
•	Create an interactive and user-friendly dashboard.
•	Analyze sales performance across months, cities, regions, and product categories.
•	Improve data storytelling using visual elements.
•	Practice real-world data analytics concepts.

## ✅ KPI Questions (List Form)
1.	What is the total sales revenue?
2.	Which month has the highest sales?
3.	Which month has the lowest sales?
4.	What is the monthly sales growth rate?
5.	Which product category generates maximum sales?
6.	Which product has the lowest sales contribution?
7.	What is the sales percentage share by product?
8.	Which region performs best in sales?
9.	Which region has the lowest performance?
10.	What is the sales distribution across regions (North, South, West)?



Overall, this project showcases practical skills in data cleaning, data analysis, dashboard design, and visualization using Excel, making it a valuable solution for business reporting and performance monitoring.
