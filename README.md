## 🚀 Project Overview
1.	Convert raw data into actionable insights
2.	Create an interactive and user-friendly dashboard
3.	Analyze sales performance across months, cities, regions, and product categories
4.	Improve data storytelling using visual elements
5.	Practice real-world data analytics concepts.

2.	Which month has the highest sales?
3.	Which month has the lowest sales?

5.	Which product category generates maximum sales?
